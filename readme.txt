= mm2csv.py =
* FreeMindで作成したマインドマップをCSVファイルに変換します
* テストケースの変換を意図したスクリプトです

= 使い方 =
* コマンドラインから実行します
* 第1引数にファイル名を指定してください
    * 例: $ python mm2csv.py sampleTestCase.mm
* スクリプトファイルと同じディレクトリにCSVファイルが作成されます
* テストケースの期待値がマインドマップのリーフノードに書かれることを想定しています

= その他 =
* えいやっと作ったのでXMLツリーをCSVに変換する機能しかありません
    * エラー処理とかもしてません
    * 機能追加を予定しています
        * アイコンの判別(×だったら無視とか)
        * 指定した部分木のみcsvに変換
